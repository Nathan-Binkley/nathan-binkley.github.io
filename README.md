# Nathaniel Binkley - Personal Portfolio

Personal portfolio website built with Next.js, React, and Mantine v5.10.

## Features

- **Hero Section**: Professional introduction with headshot and resume download
- **Daily Quotes**: Rotating quotes that change based on the day of the year
- **Projects**: Highlighted projects including Twanswatwr and other notable work
- **Contact Section**: Easy access to contact information and social links

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Mantine v5.10
- Tabler Icons

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Building for Production

```bash
npm run build
```

This will create an `out` folder with static files ready for deployment.

## Deployment

This site is configured for GitHub Pages deployment with static export. The `CNAME` file is set up for the custom domain `nathan-binkley.com`.
